Your 3DS becomes a wireless camera for your visuals: the image streams live into any Spout-compatible software, with the touch screen as a remote control.

Quick start

Installation en 5 étapes

Setup in 5 steps

Aucun fichier à éditer. Une seule chose à configurer : l'IP du PC, directement sur la 3DS.

No files to edit. Only one thing to set up: the PC's IP, right on the 3DS.

1

Installer sur le PC — tout est automatique

Install on the PC — fully automatic

Télécharge la release, dézippe, double-clique. Packages Python et Spout s'installent tout seuls.

Download the release, unzip, double-click. Python packages and Spout install themselves.

install\INSTALL.cmd
2

Copier l'app sur la carte SD (une fois)

Copy the app to the SD card (once)

Copie 3ds-cam-stream.3dsx dans le dossier 3ds de la carte SD, remets-la dans la console.

Copy 3ds-cam-stream.3dsx into the SD card's 3ds folder, put it back in the console.

3

Lancer le bridge sur le PC

Start the bridge on the PC

La fenêtre affiche IP PC pour la 3DS : 192.168.x.x — note ce numéro. Autorise le pare-feu si demandé.

The window shows IP PC pour la 3DS: 192.168.x.x — note that number. Allow the firewall prompt if asked.

install\run-bridge.cmd
4

Entrer l'IP sur la 3DS

Enter the IP on the 3DS

Homebrew Launcher → 3DS2SPOUT → bouton IP PC → tape le numéro → OK
(enregistré sur la console — à refaire seulement si l'IP du PC change)

Homebrew Launcher → 3DS2SPOUT → PC IP button → type the number → OK
(saved on the console — only needed again if the PC's IP changes)

5

Source Spout + REC

Spout source + REC

Dans ton logiciel : ajoute la source Spout 3DS2SPOUT. Sur la 3DS : REC — l'écran affiche PC OK + FPS.

In your software: add the 3DS2SPOUT Spout source. On the 3DS: REC — the screen shows PC OK + FPS.
